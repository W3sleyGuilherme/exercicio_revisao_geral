Sistema Binario

O sistema binário é um sistema numérico que utiliza apenas dois dígitos, 0 e 1. Cada dígito binário é chamado de bit. A representação de números binários segue a base 2, onde cada posição em um número binário representa uma potência de 2. Por exemplo, o número binário 1011 representa 1x2^3 + 0x2^2 + 1x2^1 + 1x2^0, que é equivalente ao número decimal 11.

O que São Valores Booleanos?

Valores booleanos são um tipo de dado que pode ter apenas dois valores: verdadeiro (true) ou falso (false). Esses valores são fundamentais na lógica booleana e são comumente usados em programação para expressar condições lógicas, tomadas de decisão e avaliações de veracidade. Em contextos de programação, operadores lógicos como AND, OR e NOT frequentemente manipulam valores booleanos para controlar o fluxo de execução do código.

Exemplos

1. Igualdade: `5 == 5` (verdadeiro)
2. Desigualdade: `3 != 7` (verdadeiro)
3. Maior que: `10 > 8` (verdadeiro)
4. Menor que: `4 < 2` (falso)
5. Maior ou igual a: `6 >= 6` (verdadeiro)
6. Menor ou igual a: `9 <= 3` (falso)
7. Combinação lógica: `(3 < 5) && (8 == 8)` (verdadeiro, devido ao uso do operador AND)
8. Negativa: `!(4 > 2)` (falso, devido ao operador NOT)

Conceito de Strings

Strings são sequências de caracteres, como letras, números e símbolos, utilizadas para representar texto em programação. Em geral, são delimitadas por aspas simples ou duplas. Strings permitem manipulação e processamento de informações textuais em linguagens de programação.

Exemplo

Pythom:

string1 = "Olá, "
string2 = "mundo!"
concatenada = string1 + string2
print(concatenada)

O que é um Array?

Um array (ou lista) é uma estrutura de dados que armazena elementos do mesmo tipo em posições sequenciais de memória. Sua utilidade reside na capacidade de organizar e acessar dados de maneira eficiente, permitindo a manipulação de conjuntos de valores em programação. Arrays facilitam o armazenamento e recuperação de informações relacionadas, melhorando a eficiência na gestão de dados em diferentes contextos de desenvolvimento de software.

Exemplo

# Declarando um array
meu_array = [1, 2, 3, 4, 5]

# Acessando elementos por índice
primeiro_elemento = meu_array[0]
segundo_elemento = meu_array[1]

print(primeiro_elemento) # Saída: 1
print(segundo_elemento) # Saída: 2

O que são estruturas condicionais?

Estruturas condicionais são construções de controle de fluxo em programação que permitem que o código execute diferentes blocos de instruções com base em condições específicas. Geralmente, a estrutura condicional avalia uma expressão booleana e, dependendo se ela é verdadeira ou falsa, direciona a execução para diferentes caminhos. As estruturas condicionais comuns incluem `if` (se), `else` (senão) e `else if` (senão se), permitindo que o programa faça escolhas lógicas e tome decisões dinâmicas com base nas condições fornecidas.

Exemplo

idade = 18

if idade >= 18:
print("Você é maior de idade.")
else:
print("Você é menor de idade.")

O que são loops?

Loops são estruturas de controle de fluxo em programação que permitem a repetição de um bloco de código enquanto uma condição específica é atendida. Essas estruturas são essenciais para automatizar tarefas repetitivas e iterar sobre conjuntos de dados. Os loops mais comuns incluem o "for", que itera sobre uma sequência, como uma lista, e o "while", que executa o bloco de código enquanto uma condição é verdadeira. Loops são fundamentais para melhorar a eficiência e a concisão do código.

Exemplos

Loop "for" para iterar sobre uma lista:

frutas = ["maçã", "banana", "laranja"]

for fruta in frutas:
print(fruta)

Loop "while" para contar até 5:

contador = 1

while contador <= 5:
print(contador)
contador += 1

conceito de funções

Funções são blocos de código encapsulados que executam uma tarefa específica e podem ser chamados (invocados) em diferentes partes de um programa. Elas ajudam a organizar o código, promovendo a modularidade e reutilização. Uma função geralmente recebe argumentos como entrada, realiza operações internas e, opcionalmente, retorna um resultado. Em muitas linguagens de programação, as funções são declaradas com um nome, parâmetros e um corpo de código, facilitando a manutenção e compreensão do programa.

Exemplo

def saudacao(nome):

"""Esta função imprime uma saudação personalizada."""
print("Olá, " + nome + "! Como você está?")

Chamando a função:

saudacao("Alice")

O que são variáveis e para que servem?

Variáveis são espaços de armazenamento nomeados que permitem atribuir valores e referenciar dados em programas de computador. Elas têm a função de armazenar e representar informações que podem ser manipuladas durante a execução do código. As variáveis facilitam a manipulação de dados, tornando o código mais flexível e dinâmico. Elas são utilizadas para armazenar números, texto, objetos e outros tipos de dados, permitindo que os programadores gerenciem informações de forma eficiente e interajam com diferentes partes do programa.

Exemplo

# Declaração de variáveis
idade = 25
nome = "João"
altura = 1.75
tem_permissoes = True

# Utilização das variáveis
print("Nome:", nome)
print("Idade:", idade, "anos")
print("Altura:", altura, "metros")

if tem_permissoes:
print(nome, "tem permissões especiais.")
else:
print(nome, "não tem permissões especiais.")

operadores comuns em programação

Operadores Aritméticos:

  • + (adição)
  • - (subtração)
  • * (multiplicação)
  • / (divisão)
  • % (módulo - resto da divisão)
  • ** (exponenciação)

Operadores Relacionais:

  • == (igual a)
  • != (diferente de)
  • < (menor que)
  • > (maior que)
  • <= (menor ou igual a)
  • >= (maior ou igual a)

Operadores Lógicos:

  • and (E lógico)
  • or (OU lógico)
  • not (NÃO lógico)

Exemplos de Operadores Aritméticos:

a = 10
b = 3

soma = a + b # 13
subtracao = a - b # 7
multiplicacao = a * b # 30
divisao = a / b # 3.3333333333333335
resto = a % b # 1
exponenciacao = a ** b # 1000

Exemplos de Operadores Relacionais:

x = 5 y = 8 igual = x == y # False diferente = x != y # True menor_que = x < y # True maior_que = x > y # False menor_ou_igual = x <= y # True maior_ou_igual = x >= y # False

Exemplos de Operadores Lógicos:

p = True
q = False

E_logico = p and q # False
OU_logico = p or q # True
NAO_logico = not p # False
